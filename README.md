# SellerMargin — Etsy Fee & Profit Calculator (2026)

Free, fully client-side fee & profit calculator for Etsy sellers, updated for
the **June 22, 2026 regulatory fee changes**. Covers US, UK, CA, AU, DE, FR,
IT, ES, NL — including VAT charged on seller fees, shipping-inclusive Offsite
Ads, and CA/AU international processing rates.

**Live:** https://hage-inkyat.github.io/sellermargin/

Nothing you type leaves your browser. Every rate is sourced from Etsy's
official fee documentation — see the Sources section on the page, or
`js/fees-data.js` (rates + source URLs + verification date).

Found a stale rate? Open an issue with the official Etsy source link.
